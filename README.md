# 🎮 Super Pikachu Bros - World 1-1

一個以超級瑪莉為靈感的復古風網頁遊戲，使用皮卡丘作為主角。完整實現了經典的 1-1 關卡，包含投幣機制、平台跳躍、敵人互動和道具收集。

[Demo URL](https://gistshu.github.io/MarioWeb/)

## 🌟 特色功能

- 🪙 **經典投幣系統** - 復古街機風格的開始畫面
- 🎨 **復古像素美學** - 8-bit 風格的視覺設計
- 🎮 **流暢的遊戲機制** - 60 FPS 遊戲循環
- 🏃 **完整的物理系統** - 重力、跳躍、碰撞檢測
- 👾 **經典敵人** - 蘑菇型敵人（類似栗寶寶）
- 🪙 **金幣收集** - 可收集的金幣和問號方塊
- 📱 **響應式設計** - 支援桌面、平板和手機

## 🎯 遊戲目標

操控皮卡丘穿越 1-1 關卡，收集金幣、擊敗敵人，最終到達終點旗桿！

## 🎮 操作說明

### 菜單畫面
- **X 鍵** - 投幣（增加硬幣數量）
- **空白鍵** - 開始遊戲（需要至少 1 個硬幣）

### 遊戲中
- **方向鍵 ←→** - 左右移動
- **A 鍵** - 跳躍
- **B 鍵** - 加速奔跑（配合方向鍵使用）

## 🚀 快速開始

### 方法一：直接開啟（推薦）

1. 下載或克隆此專案
2. 直接在瀏覽器中開啟 `index.html` 檔案

### 方法二：使用本地伺服器

```bash
# 使用 Python 簡易伺服器
python -m http.server 8080

# 或使用 Node.js http-server
npx http-server -p 8080
```

然後在瀏覽器中訪問 `http://localhost:8080`

### 方法三：GitHub Pages（線上遊玩）

本專案已配置為可直接部署到 GitHub Pages：

1. Fork 此專案到您的 GitHub 帳號
2. 進入 Settings > Pages
3. Source 選擇 `main` 分支和 `/ (root)` 資料夾
4. 點擊 Save
5. 幾分鐘後即可通過提供的 URL 訪問

## 🎨 技術特點

### 前端技術
- **HTML5 Canvas** - 遊戲渲染
- **原生 JavaScript (ES6+)** - 無框架依賴
- **CSS3** - 動畫與響應式設計
- **Google Fonts** - Press Start 2P 復古字體

### 遊戲引擎功能
- ✅ 自訂物理引擎（重力、速度、加速度）
- ✅ AABB 碰撞檢測系統
- ✅ 側向捲軸相機系統
- ✅ 動畫系統
- ✅ 狀態管理（選單、遊戲中、暫停、結束）
- ✅ 程序生成像素藝術

### 遊戲機制
- **跳躍物理** - 仿超級瑪莉的跳躍手感
- **敵人 AI** - 自動巡邏與轉向
- **可破壞方塊** - 從下方撞擊可破壞磚塊
- **互動問號方塊** - 撞擊後掉落金幣
- **計分系統** - 金幣收集、擊敗敵人加分
- **時間限制** - 倒數計時器

## 🎯 遊戲提示

1. **跳躍控制** - 按住跳躍鍵可跳得更高
2. **擊敗敵人** - 從上方跳到敵人頭上
3. **破壞方塊** - 從下方撞擊磚塊
4. **問號方塊** - 撞擊後會給予金幣（每個只能用一次）
5. **加速奔跑** - 按住 B 鍵可以跑得更快、跳得更遠
